# MedShop (v1.0 - Legacy)

> **Notice:** This repository represents the original MedShop v1.0 architecture. It is preserved here for historical context and portfolio demonstration. Active development has moved to the **MedShop-2.0** repository.

## Architecture Overview
MedShop 1.0 is an N-Tier ASP.NET Core 6.0 e-commerce platform featuring:
* **UI:** Server-side rendered MVC Razor Views with Bootstrap 5.
* **API:** Asynchronous REST endpoints consumed via jQuery.
* **Data:** Entity Framework Core with generic repository patterns.
* **Features:** Role-based Identity, session-based asynchronous shopping cart, and custom decimal model binding.
